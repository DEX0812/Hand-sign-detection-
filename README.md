# ASL Hand Sign Recognition

This project uses MediaPipe and OpenCV to recognize ASL hand signs in real-time.

## Project Structure
- `src/`: Contains the source code.
- `models/`: Contains the hand landmarker model.
- `requirements.txt`: List of dependencies.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. **Launch Application**: Run `START_APP.bat` (Opens in your browser)
3. **Stop Application**: Run `scripts\STOP_APP.bat`

## Features
- Real-time hand sign recognition.
- 90%+ accuracy target using rule-based and temporal smoothing.
- Visual feedback with hand landmark overlays.
- Sentence building capabilities.
